================================================================================
                              HOW TO USE
================================================================================

1. OPEN APPLICATION
   - Open index.html in your browser

2. ADD ENTRIES
   - Income form now has date field
   - Expense form now has date field
   - Dates default to today

3. EXPORT DATA
   - Scroll to Data Management section
   - Set date range (optional)
   - Click desired export button
   - File downloads automatically

4. IMPORT DATA
   - Click "Import from CSV"
   - Select CSV file
   - Map columns (auto-detected)
   - Review validation
   - Confirm import

5. EXPORT TO PDF
   - Click "Export to PDF"
   - Browser print dialog opens
   - Select "Save as PDF"
   - Choose location and save
